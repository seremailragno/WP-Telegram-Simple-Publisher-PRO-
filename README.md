WP Simple Telegram Publisher PRO is a lightweight and professional WordPress plugin that lets you automatically publish your articles on Telegram, with full control over:

- Immediate article delivery (FREE feature)
- Integrated connection test (FREE feature)
- Publishing time (PRO feature)
- Time zone (PRO feature)
- Resend when article is updated (enabled/disabled, PRO feature)
- Scheduled delivery (PRO feature)
- Exclusion/inclusion based on tags/categories (PRO feature)

  It is designed for bloggers, online newspapers, creators, and news sites who want to reliably automate content distribution on Telegram.

How to configure the plugin:
1. Create a Telegram Bot
- Open Telegram
- Search for @BotFather
- Type /newbot
- Follow the instructions
- Copy the Bot Token

2. Retrieve Chat ID
- Add the bot to your channel as an administrator
- Send a message
- Open in browser: https://api.telegram.org/botTUO_TOKEN/getUpdates
- Search for "chat":{"id":…}
- Copy the value

3. Enter the data into the plugin
- Bot Token  
- Chat ID  
- Time (PRO feature)
- Timezone (PRO feature)
- Enable or disable resending updates (PRO feature)
- Save and test.
